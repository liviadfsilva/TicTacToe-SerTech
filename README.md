# Tic-Tac-Toe 💚 Ser+ Tech

![Java](https://img.shields.io/badge/Java-21-orange?style=for-the-badge)

## Sobre 🚀
Este é um jogo de Tic-Tac-Toe, ou Jogo da Velha, é um jogo de linha de comando desenvolvido como projeto para o programa Ser+ Tech. O objetivo foi praticar conceitos fundamentais do Java, como por exemplo:

- Arrays
- Estruturas de controle
- Lógica básica de jogos
- Tratamento de exceções
- Entrada de dados com Scanner

<br>

## Instalação 💻

1. **Clone o repositório:**

   ```sh
   git clone https://github.com/liviadfsilva/TicTacToe-SerTech.git
   ```

2. **Acesse a pasta do projeto:**
   
   ```sh
    cd TicTacToe-SerTech
   ```

3. **Compile o programa:**
   
   ```sh
    javac App.java
   ```

4. **Execute:**
   
   ```sh
    java App
   ```

<br>

## Como Jogar 🕹️
1. Execute o programa

2. Os jogadores se revezam digitando:

```
linha
(aperte enter)
coluna
```

Exemplo:

```
1
2
```

3. Os valores devem estar entre `0` e `2`  

4. O jogo termina quando:
- Um jogador vence  
- O tabuleiro fica cheio (empate)

<br/>

## Regras de Entrada ⚠️

- Apenas números são aceitos
- Os valores devem estar entre `0` e `2`
- Não é permitido jogar em uma posição já ocupada
